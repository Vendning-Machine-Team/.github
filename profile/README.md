# Vending Machine Robot

### By Samantha Machado, Prince Patel, Matthew Beck, Syan Shirazi, Samuel Saylor, Cayden Hutcheson, Tri Nguyen, Adeyemi Akanbi, Ryler Collins, Victoria Adams, and Anthony Nguyen

**Please consider:** if you like it, **star our repositories!**

## Overview
Initially starting out as a joke, we asked the question: "What if instead of walking up to a vending machine, the vending machine rolled up to you?"

This is what happens when jokes go too far. Enjoy!
